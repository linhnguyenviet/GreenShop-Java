### GREEN SHOP - JAVA
We use Java to create a MVC pages about e-commerce. 

## Technology : 
+ Java JSP/Servlet
+ HTML/CSS/JavaScript
+ SQL Server

## Features : 
+ Sign Up, Sign In
+ Buying Plants 
+ Manage Orders
+ Admin management for shop
+ Comment, feedback

## Using guides :
+ Select what you want, add to your cart.
+ Then sign in ( or sign up if you dont have an account yet ) and checkout. Happy shopping ^^
